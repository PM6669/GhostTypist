GhostTypist is a hidden PowerShell script that randomly types messages in the currently focused window, simulating a "ghost" using your keyboard.

## Features
- Runs hidden in background
- Single instance only
- Human-like typing speed
- Random delays and sentences
- Works in any text field

## How to Use

1. Download the project
2. Double click `StartGhost.bat`
3. Open Notepad or any text box
4. Click inside it and wait...

To stop it: open Task Manager and kill `powershell.exe`.
